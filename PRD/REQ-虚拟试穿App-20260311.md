## 一、产品概述

### 1.1 背景与问题

目标用户在电商平台购买服装时，面临以下痛点：

- **试穿成本高**：批量购买多件 → 试穿 → 退换，耗时耗力
- **决策困难**：难以判断商品是否适合自己的风格、身材
- **缺乏参考**：家人/朋友无法实时给意见
- **记忆模糊**：难以记住什么风格适合自己，重复踩坑

### 1.2 产品定位

一款**虚拟试穿 + AI 穿搭助手**应用，帮助用户在购买前预览搭配效果，辅助决策。

### 1.3 目标用户

| 用户群体 | 特征              |
| ---- | --------------- |
| 核心用户 | 18-40岁女性，电商购物频繁 |
| 扩展用户 | 男性、儿童、宠物主人      |

### 1.4 使用场景

- 用户在家浏览电商平台，看到心仪商品
- 截图商品图片，导入 App 进行虚拟试穿
- 查看 AI 多维度点评，获取穿搭建议
- 决策是否购买，或分享给亲友征求意见

### 3.2 功能模块详细说明

#### 模块一：形象管理

| 功能点    | 说明                   | MVP  |
| ------ | -------------------- | :--: |
| 拍照上传形象 | 用户拍摄/上传全身照作为试穿底图    |  ✅   |
| 照片管理   | 查看和管理已上传的形象照片       |  ✅   |
| 多形象管理  | 创建多个形象（不同场景/角色）      | 后期   |
| 形象更新   | 随时间更新形象照片            |  ✅   |

**用户流程：**
1. 用户点击「创建形象」
2. 拍摄或上传全身照
3. 系统保存照片作为试穿底图
4. 用户可为形象命名
5. 保存形象

**技术说明：**
- MVP 阶段采用用户真实照片作为试穿底图，无需生成 3D 形象
- 降低技术复杂度，快速验证核心流程
- 后期可迭代升级为 3D 形象方案

---

#### 模块二：虚拟试穿

| 功能点   | 说明                     | MVP  |
| ------- | ------------------------ | :--: |
| 截图导入  | 从相册选择电商截图           |  ✅   |
| 服装裁剪  | 手动框选截图中的服装区域       |  ✅   |
| 图层叠加  | 将裁剪后的服装叠加到用户照片上   |  ✅   |
| 调整编辑  | 调整服装位置、大小、旋转、透明度 |  ✅   |
| 效果合成  | 生成最终试穿效果图           |  ✅   |
| 多件对比  | 同时试穿多件进行对比         | 后期   |

**用户流程：**
1. 用户在电商平台看到心仪商品
2. 截图保存到相册
3. 在 App 中选择形象照片
4. 导入服装截图
5. 手动框选服装区域进行裁剪
6. 在编辑器中调整服装位置、大小、角度
7. 生成并保存试穿效果图

**技术方案（低成本方案）：**

采用**图层叠加**方式实现虚拟试穿，核心原理：

```
┌─────────────────────────────────────────┐
│              试穿编辑器                   │
├─────────────────────────────────────────┤
│  底层：用户形象照片（固定）                 │
│  ┌─────────────────────────────────┐    │
│  │                                 │    │
│  │         用户全身照片              │    │
│  │                                 │    │
│  └─────────────────────────────────┘    │
│                                         │
│  上层：服装图层（可调整）                  │
│  ┌───────────┐                          │
│  │  裁剪后的  │  ← 拖拽移动              │
│  │  服装图片  │  ← 双指缩放              │
│  │           │  ← 旋转调整              │
│  └───────────┘  ← 透明度调节            │
│                                         │
└─────────────────────────────────────────┘
```

**编辑参数：**

| 参数     | 说明           | 范围        |
| -------- | -------------- | ----------- |
| offsetX  | X轴偏移        | -∞ ~ +∞    |
| offsetY  | Y轴偏移        | -∞ ~ +∞    |
| scale    | 缩放比例       | 0.1 ~ 3.0  |
| rotation | 旋转角度（弧度）| -π ~ +π    |
| opacity  | 透明度         | 0.1 ~ 1.0  |

**效果预期：**
- 风格预览为主，类似"换装小游戏"体验
- 用户可直观看到服装款式、颜色搭配效果
- 支持快速切换对比不同服装

**数据模型：**

```

├── aiRating: AI评分(0-100)
├── aiComment: AI点评内容
└── createdAt: 创建时间

用户偏好画像(UserPreference)
├── userId: 用户ID
├── topStyles: 常穿风格Top5
├── colorPreferences: 高频颜色[]
├── occasionDistribution: 场合分布{}
├── bodyFeatures: 身材特征（可选）
└── updatedAt: 更新时间
```

## 六、技术方案

### 6.1 技术架构

```
┌─────────────────────────────────────────────────────────────┐
│                        Flutter App                          │
├─────────────────────────────────────────────────────────────┤
│  ┌──────────┐  ┌──────────┐  ┌──────────┐  ┌──────────┐    │
│  │ 形象管理  │  │ 试穿编辑器│  │ 服装裁剪  │  │ 效果保存  │    │
│  └──────────┘  └──────────┘  └──────────┘  └──────────┘    │
│       │              │              │              │        │
│       ▼              ▼              ▼              ▼        │
│  ┌─────────────────────────────────────────────────────┐   │
│  │              本地存储 (Hive)                         │   │
│  └─────────────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────┐
│                      Spring Boot 后端                        │
├─────────────────────────────────────────────────────────────┤
│  用户账户 │ 数据同步 │ AI 点评 │ 文件存储 (OSS)              │
└─────────────────────────────────────────────────────────────┘
```

**核心特点：**
- 试穿合成完全在客户端本地完成，**零服务器成本**
- 后端仅处理账户、数据同步、AI 点评
- 支持离线使用基础功能

### 6.2 客户端技术栈

| 类别     | 技术选型               | 说明           |
| -------- | ---------------------- | -------------- |
| 框架     | Flutter 3.x            | 跨平台，一套代码 |
| 语言     | Dart                   |                |
| 状态管理 | Riverpod               | 响应式状态管理  |
| 本地存储 | Hive                   | 轻量级 NoSQL   |
| 图像选择 | image_picker           | 拍照/相册      |
| 图像裁剪 | image_cropper          | 手动框选裁剪   |
| 图像处理 | image (Dart 库)        | 图片合成       |

### 6.5 技术方案演进路线

```
MVP v1（第1-4周）

└─ 成本：¥0

├─ 试穿：加入姿态估计（MediaPipe）

├─ 形象：集成 Ready Player Me 3D 形象
```
